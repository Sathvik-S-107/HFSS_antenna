# HFSS 5.2 GHz Meandered Monopole Antenna Design

## Project Overview

This project presents the design and simulation of a compact 5.2 GHz meandered monopole antenna for WLAN applications using Ansys HFSS. The antenna was designed on an FR4 substrate and optimized for operation in the 5 GHz WLAN band.

The design focuses on achieving good impedance matching, compact size, and acceptable radiation characteristics for wireless communication applications.

---

## Software Used

* Ansys HFSS
* FR4 Substrate Material

---

## Design Specifications

| Parameter           | Value                         |
| ------------------- | ----------------------------- |
| Operating Frequency | 5.2 GHz                       |
| Antenna Type        | Meandered Monopole            |
| Feed Method         | Microstrip Feed               |
| Substrate           | FR4                           |
| Application         | WLAN / Wireless Communication |

---

## Design Methodology

1. Designed the FR4 substrate geometry.
2. Developed a compact meandered monopole radiator structure.
3. Implemented microstrip feed excitation.
4. Created radiation boundary conditions in HFSS.
5. Performed frequency sweep analysis.
6. Optimized antenna dimensions to achieve resonance near 5.2 GHz.
7. Evaluated return loss and radiation characteristics.

---

## Simulation Results

### Return Loss (S11)

* Resonant Frequency ≈ 5.2 GHz
* Simulated S11 ≈ -13 dB

The antenna achieved good impedance matching within the desired WLAN operating band.

### Radiation Characteristics

The antenna radiation performance was evaluated using:

* E-Plane Radiation Pattern
* H-Plane Radiation Pattern

The results demonstrate suitable radiation behavior for wireless communication applications.

---

## Experimental Validation

The fabricated antenna was tested and compared with simulation results.

### Measured Results

* Measured resonance observed near 5 GHz WLAN band.
* Measured S11 reached approximately -24 dB.
* Experimental results showed good agreement with simulation.

---

## RSSI Performance Evaluation

Received Signal Strength Indicator (RSSI) measurements were conducted for different antenna configurations.

Observations:

* Proposed antenna outperformed the no-antenna configuration.
* Performance was comparable to commercial antenna solutions.
* Reliable communication was maintained across increasing distances.
